# cd2date-ng v1.0
Basic reverse engineering of cd2date by Aztekk. Should work identical to v2.0B1
Original can be found [here](https://archive.org/details/cd2date_v2.0B1).

	Usage: ./cd2date-ng <manufacturer> <hubcode> [decade]  
	Manufacturers:  
	CMC Magnetics, Daxon Technology, Ritek Corporation, MJC Pte Ltd (Singapore),  
	Optodisc Technology, TDK Corporation (Luxembourg), Mitsubishi Chemical Corporation [CD-R],  
	Mitsubishi Chemical Corporation [DVD-R], Taiyo Yuden, Mitsui Toatsu Chemicals + MAM-E,  
	Moser Baer India, Prodisc, Eastman Kodak Company  
	  
	Note: Decade defaults to 2000 if not specified (used for standard YDDD format).
